# [News Pad](https://github.com/qvil/newspad)

Mobile News App created by react-native

API Reference : https://newsapi.org/

# Demo

![demo.gif](/demo.gif)

# Install

```
npm install
```

# Run

## Prerequisite

* [Expo](https://expo.io/)
* /config/config.json (It is not managed by git, Get your API Key https://newsapi.org/)

```json
{
  "apiKey": "YOUR_API_KEY"
}
```

```
npm start
```
